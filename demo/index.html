Get Started with Drawbridge

Ready to use Drawbridge in your own projects? Follow these simple steps.

1

Install Chrome Extension

Add the Drawbridge extension to Chrome from the Chrome Web Store or load the unpacked extension from GitHub.

2

Connect Your Project

Open your local development server and connect Drawbridge to start annotating your UI elements.

3

Start Annotating

Press 'f' to enter annotation mode, click any element, and describe your desired changes. AI will handle the rest!
