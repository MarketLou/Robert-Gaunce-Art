🧭 Drawbridge Setup & Usage Guide

Learn how to set up and use Drawbridge to turn visual feedback into code changes

🚀 1. Setup

1

Clone the Drawbridge repo

Clone this repository into your project directory

2

Load the Chrome extension

Go to chrome://extensions/, enable Developer mode, and click "Load unpacked". Select the moat-chrome/ folder

3

Activate and pin the extension

Click the extension icon and pin it to your Chrome toolbar for easy access

💬 2. Making Comments

C

Press 'C' to start commenting

Your cursor will turn into a crosshair for element selection

🎯

Hover over elements

DOM elements will be highlighted as you move your cursor

✏️

Click to comment

Click an element, type your comment, and press Enter or Submit

📋

View tasks in the Moat

Your tasks appear in the bottom area of the page

🤖 3. Processing with Cursor

Run the bridge command

bridge

Processing modes:

step bridge (Default) - Process tasks one by one with approval
batch bridge - Group related tasks and process in batches
yolo bridge - Process all tasks without stopping (use carefully)

🎯 Best Practices

Be Specific

✅ "change font to sans-serif"
❌ "change font"

Chain Related Changes

✅ 1. "make button blue" → 2. "add shadow to the blue button"
❌ "make button blue and add a shadow"

One Change Per Comment

✅ Focus on one distinct change
❌ Multiple unrelated changes in one comment

🐛 Troubleshooting

"Dependency Error"

Tasks might be out of order. Check the AI's analysis for the required sequence.

Task failed status

Check moat-tasks.md. Reset status to "pending" in the .json file to retry.

Connection Issues

Press Cmd/Ctrl+Shift+P in browser and re-select your project directory.
